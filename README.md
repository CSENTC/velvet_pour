# Velvet Pour

Velvet Pour is a premium React website template designed for cocktail, beverage, and luxury product brands. Built around a fictional mojito cocktail shop, the template delivers a cinematic, scroll-driven browsing experience powered by modern web animation techniques.

The template combines advanced GSAP animations with a clean component-based React architecture to create an immersive user experience while maintaining excellent performance and responsiveness across all devices.

---

# Overview

Velvet Pour showcases modern web animation patterns through a collection of carefully crafted interactions and scroll-driven storytelling techniques. Every section is designed to provide a smooth visual experience using GSAP's animation capabilities while remaining modular and easy to customize.

Whether you're building a product landing page, beverage brand website, restaurant showcase, or promotional microsite, Velvet Pour provides a strong foundation with production-ready code and reusable components.

# Animation Highlights

## SplitText Animations

Create engaging typography using GSAP SplitText with character, word, and line-based reveal animations.

## ScrollTrigger

Drive animations directly from the user's scroll position using GSAP ScrollTrigger, enabling smooth transitions and synchronized timelines.

## Parallax Effects

Layered scrolling animations provide depth and improve the visual presentation throughout the website.

## Pinned Sections

Pin sections during scrolling to create immersive storytelling and controlled content transitions.

## Scroll-Synchronized Video

Synchronize video playback with the user's scroll position for a cinematic browsing experience.

## Image Masking

Reveal images through animated masks to create polished visual transitions.

## Custom Carousel

A fully custom-built animated carousel developed with GSAP instead of traditional slider libraries, providing smooth transitions and greater customization.

## Timeline Animations

Complex GSAP timelines coordinate animations across multiple sections, creating a seamless user experience from start to finish.

# Technology Stack

## React

React provides the component-based architecture for the project, enabling reusable UI components, maintainable code, and seamless integration with advanced animation workflows.

## GSAP

GSAP powers every animation throughout the template, including:

- SplitText animations
- ScrollTrigger interactions
- Timeline animations
- Parallax scrolling
- Pinned sections
- Scroll-controlled video playback
- Image masking effects
- Custom carousel animations

## Tailwind CSS

Tailwind CSS is used for building a modern, responsive interface with utility-first styling, allowing rapid development while maintaining consistency throughout the project.

## Vite

Vite serves as the development environment and build tool, providing fast startup times, instant Hot Module Replacement (HMR), and optimized production builds.

# Installation

Clone the repository:

```bash
git clone https://github.com/CSENTC/velvet_pour
```

Navigate to the project directory:

```bash
cd velvet-pour
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

# Production Build

Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
